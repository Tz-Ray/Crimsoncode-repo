AiTaskGroup project
TaskCalendarApp

instructions for running. 
-clone into repository on wsl/linux with https link from github
-navigate into root folder, should have two folders in the directory and a .md file and a .sh file
-in the terminal, type "chmod +x start-app.sh"
-now type "./start-app.sh" and the program should begin launching

From there you can add tasks, get AI summaries of your day/week/month plans, create study plans for specific events, and have an overall easier time managing your work in life and school.
